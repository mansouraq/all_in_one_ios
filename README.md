Key Features :

```
The main objective of this project is to develop a social platform (Android and iOS) from where users can interact with friends, view stories, listen music, view events, view live game statics, do payments, view expenses, view news, view religious information. Administrator of the platform will be able to manage the user roles and all the platform components.

```

```
Pod installation Steps --


1. Open Terminal
2. sudo gem install cocoapods <press enter>
3. pod setup <press enter>


NOTE:- Don't close terminal it'll take time for installing

```
